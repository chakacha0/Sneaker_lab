# Настройка корзины

## Шаг 1: Применить миграцию для product_stock

Выполните SQL скрипт для изменения ограничения quantity:

```sql
ALTER TABLE product_stock 
DROP CONSTRAINT IF EXISTS product_stock_quantity_check;

ALTER TABLE product_stock 
ADD CONSTRAINT product_stock_quantity_check CHECK (quantity >= 0);
```

Или выполните файл:
```bash
psql -U postgres -d shoeses_shop -f migration_fix_product_stock.sql
```

## Что реализовано:

### Backend:
1. **Репозиторий корзины** (`cart_repo.py`):
   - Создание/получение корзины пользователя
   - Добавление товаров в корзину
   - Получение всех товаров из корзины
   - Удаление товаров
   - Обновление количества

2. **Репозиторий размеров** (`product_stock_repo.py`):
   - Получение размеров товара с количеством

3. **API эндпоинты**:
   - `POST /cart/add` - добавить товар в корзину
   - `GET /cart/{user_id}` - получить корзину пользователя
   - `DELETE /cart/item/{cart_item_id}` - удалить товар
   - `PUT /cart/item/{cart_item_id}` - обновить количество
   - `GET /products/{product_id}/sizes` - получить размеры товара

### Frontend:
1. **Страница товара** (`ProductPage.jsx`):
   - Отображение размеров из базы данных
   - Серые размеры для товаров с quantity = 0
   - "Нет в наличии" вместо цены для недоступных размеров
   - Проверка выбранного размера перед добавлением
   - Проверка авторизации

2. **Страница корзины** (`Cart.jsx`):
   - Список товаров с информацией
   - Изменение количества
   - Удаление товаров
   - Общая сумма заказа
   - Кнопка "Оформить заказ" (пока показывает alert)
   - Автоматическое открытие модального окна входа, если пользователь не авторизован

3. **Header**:
   - Иконка корзины рядом с иконкой пользователя
   - Переход на страницу корзины при клике

## Как это работает:

1. **Добавление в корзину**:
   - Пользователь выбирает размер на странице товара
   - Нажимает "Добавить в корзину"
   - Если не авторизован - показывается сообщение
   - Товар добавляется в корзину пользователя

2. **Просмотр корзины**:
   - Клик на иконку корзины в Header
   - Если не авторизован - открывается модальное окно входа/регистрации
   - Отображаются все товары с информацией и выбранным размером
   - Справа показывается общая сумма

3. **Управление корзиной**:
   - Изменение количества через кнопки +/-
   - Удаление товаров
   - Автоматический пересчет общей суммы

## Важно:

- Корзина привязана к пользователю (user_id)
- Если пользователь не авторизован, он не может добавить товары в корзину
- При добавлении товара с тем же размером количество увеличивается
- Размеры с quantity = 0 отображаются серым и недоступны для выбора
