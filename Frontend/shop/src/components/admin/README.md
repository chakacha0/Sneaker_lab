# Административная панель - Структура компонентов

## Текущая структура

### Основные компоненты:
- `AdminMenu.jsx` - Боковое меню навигации
- `AdminInfo.jsx` - Секция с личной информацией администратора

### Стили:
- `adminStyles.js` - Все стили для административной панели

## План дальнейшей реструктуризации

### Модальные окна (вынести в отдельные компоненты):
1. **AddBrandModal.jsx** - Добавление бренда
2. **EditBrandModal.jsx** - Редактирование бренда
3. **DeleteBrandConfirmModal.jsx** - Подтверждение удаления бренда
4. **AddProductModal.jsx** - Добавление товара
5. **EditProductModal.jsx** - Редактирование товара
6. **DeleteProductConfirmModal.jsx** - Подтверждение удаления товара
7. **ViewStockModal.jsx** - Просмотр остатков товара
8. **AddStockModal.jsx** - Добавление количества товара

### Списки (вынести в отдельные компоненты):
1. **AdminBrandsList.jsx** - Список брендов с карточками
2. **AdminProductsList.jsx** - Список товаров с карточками

### Кастомные хуки (опционально):
1. **useAdminBrands.js** - Логика работы с брендами
2. **useAdminProducts.js** - Логика работы с товарами

## Преимущества реструктуризации:
- ✅ Легче читать и понимать код
- ✅ Легче поддерживать и отлаживать
- ✅ Возможность переиспользования компонентов
- ✅ Упрощенное тестирование
- ✅ Меньше конфликтов при работе в команде
